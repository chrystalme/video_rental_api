# video_rental_api
A NodeJS api for a video rental service

## Dependencies
  - Node
  - MongoDB
  - nodemon

# How to get up and running
  - Clone repo
  ``` 
  $git clone https://github.com/chrystalme/video_rental_api.git
  ```
  - Install dependencies
  ```
  $npm install
  ```
  -Run application
  ```
  $nodemon
  ```
